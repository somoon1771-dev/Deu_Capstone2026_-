3. DB 파일 확인
- capstone_schema.sql은 CREATE TABLE 중심의 스키마 파일입니다.
- INSERT INTO 구문은 포함되어 있지 않습니다.
- DB 비밀번호, API Key, ngrok, tailscale 인증정보는 포함되어 있지 않습니다.

4. 민감정보 처리
- application.properties 제거
- application-example.properties만 포함
- ApiKeys.java는 환경변수 기반으로 변경
- 실제 DB 비밀번호/API Key 하드코딩 제거

5. 제거한 파일 유형
- 로그 파일
- .dart_tool
- .gradle
- local.properties
- APK/중복 ZIP 빌드 산출물
